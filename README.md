# Alphabet Square Pattern in C++

This is a beginner-friendly C++ pattern printing program that prints an alphabet square using nested loops.

## Features
- Takes square size input from user
- Prints alphabet square pattern
- Uses nested loops (`for loop`)
- Uses ASCII value conversion
- Typecasting from integer to character

## Example

Input:
4

Output:
ABCD
ABCD
ABCD
ABCD

## Concepts Used
- Nested Loops
- ASCII Table
- Type Casting
- Pattern Printing
- User Input (`cin`)
